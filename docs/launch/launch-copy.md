## LinkedIn Post

I just open-sourced a project called **HTML Carousel Reel**.

The idea is simple: many creators already use AI tools to generate carousels in HTML, but the final output still needs to become a proper short-form video.

This repo takes that HTML, compiles it into a local render pipeline, adds motion, staggered reveals, counters, ambient glow, and exports a vertical MP4.

What I wanted from the workflow:

- local rendering
- no paid video API
- reusable HTML as the creative source
- fast iteration with prompt-based motion tuning

If you work with carousels, reels, or AI-assisted content workflows, this may be useful.

Repo: https://github.com/radhakishan404/html-carousel-reel

Would appreciate a star, feedback, or sample ideas.

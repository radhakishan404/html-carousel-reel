04 / 09
Step 3 render video
One command

Run one command. Get MP4 output.

Renderhtml -> reel
./render-reel ./html-carousel/create_html_to_reel_carousel.html \
  --prompt "cinematic, smooth, presentation" \
  --seconds-per-slide 5.2 \
  --quality high
Voila: MP4 gets saved beside your HTML with _reel.mp4.
Output / smooth motion, line reveals, counters, cinematic glow

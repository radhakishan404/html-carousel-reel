04 / 09
Step 3 render video
One command

Run one render command. Get MP4 output.

Renderhtml -> reel
./render-reel ./html-carousel/create_html_to_reel_carousel.html \
  --prompt "cinematic neon, bold, fast" \
  --seconds-per-slide 3.4 \
  --quality high
Voila: your MP4 is saved beside the HTML file with _reel.mp4.
Output / motion-designed reel with counters, text reveals, and light sweeps

What is ghgrab
@rk.codex

A terminal tool for grabbing GitHub files without cloning.

`ghgrab` lets you browse a repo, fuzzy-search files, preview content, and pull only the paths you want. It also supports release downloads and machine-friendly agent mode.

Commands That Matter
@rk.codex

These are the commands most devs will actually use.

ghgrab https://github.com/rust-lang/rust --cwd --no-folder
Download directly into the current directory without creating a repo folder.
ghgrab rel sharkdp/bat --extract
Pull a release asset instead of browsing the repo manually.
ghgrab agent tree https://github.com/rust-lang/rust
Get a stable JSON tree for scripts, agents, CI, or automation tooling.

05 / 08
Useful Flags + Features
@rk.codex

The hidden value is in the small flags.

--cwd
Force downloads into the current working directory.
--no-folder
Skip the extra repo folder when you only need one file fast.
--token
Use an explicit GitHub token for that run without saving it locally.
⌕
Repo discovery mode
Type a repo keyword from home and browse GitHub repos instantly.
⚙
Agent mode
`agent tree` and `agent download` return machine-friendly output for automation.
